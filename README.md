vim-files
=========

This is my current vim setup for vim-gnome and Ubuntu 14.04.

Setup:

Type in your home folder:

```
git clone https://github.com/vinicarlos56/vim-files.git .vim && cd .vim && sudo sh install.sh

```

Modules used:

* nerdtree
* multiple-cursors
* vim-sensible
* linters
* tcomment_vim
* ctrlp.vim
* vim-surround
* snipmate.vim
* zencoding-vim
* tabline
* vim-airline
* vim-fugitive
* vdebug
* vim-coffee-script
